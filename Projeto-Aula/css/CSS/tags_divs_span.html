Div são feitas para estruturar o site. Por padrão ela ocupa todo o espaçamento da tela.
Utiliza-se ID e class para dar cor e formatações ao texto
É basicamente uma DIV. É um elemento do tipo INLINE style, os conteudos ficam um ao lado do outro, já o DIV fica um abaixo do outro.
Border = Borda, pode-se utilizar em todos os elementos e é necessário 3 elementos no codigo:
    A espessura (px / % / etc)
    solid/outset/dotted/dashed etc (ver mais no css bordes)
    por fim a cor da borda, que dá pra colocar 4 cores diferentes no sentido horario
    None = sem bordas
    tem mais uma maneira para ter um maior controle: width = espessura
    border-style para definir o estilo da borda
